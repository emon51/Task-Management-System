# Task Manager

A simple task management application built with React that allows users to view, search and manage tasks.

## Live Demo

[View Live Site](https://task-management-system-w3.netlify.app/)

## Features

- View list of tasks
- Search tasks by title
- Pagination (20 tasks per page)
- Toggle task status (Pending <==> Completed)
- Dark/Light mode theme
- Responsive design

## Technologies

- React
- React Router
- Context API
- CSS3
- JSONPlaceholder API

## Installation
### 1. Clone repository
```
git clone https://github.com/emon51/Task-Management-System.git
```

### 2. Navigate to project
```
cd taskmanager
```

### 3. Install dependencies
```
npm install
```

### 4. Run development server
```
npm start
```

### 5. Open this to view in browser
```
http://localhost:3000
``` 

## Usage

- **Home**: Welcome page
- **Tasks**: View all tasks with search and pagination
- **Task Details**: Click any task to view details
- **Theme Toggle**: Click moon/sun icon in navbar
- **Status Toggle**: Click status badge to toggle Pending/Completed

## Build for Production
```
npm run build
```
